Next.js 16 brings a handful of breaking changes. Here are the ones I hit first.

## `params` is a Promise

Dynamic route params are now async. You `await` them:

```tsx
export default async function Page({
  params,
}: {
  params: Promise<{ slug: string }>
}) {
  const { slug } = await params
  return <div>{slug}</div>
}
```

## Turbopack is the default

Dev and build both run on Turbopack by default. No extra config.

## Read the docs in your `node_modules`

Your training data probably predates some of these changes. The canonical
reference sits right in `node_modules/next/dist/docs/`. Read it before
guessing.
